CSS layouts

CSS Grid er et rutenett som gir struktur når man skal designe. Css grid formes rundt to akser. Grid blir brukt for å dele siden etter det visuelle. Når man bruker Grid må man da altså ikke designe etter hvordan strukutren skal være på nettsiden. Når man bruker grid så behøver man heller ikke være redd for overlapping. Når man skal bruke CSS flexbox så må man tenke at det er to akser. De to aksene kan bli delt opp i fire igjen som er "rad" " rad-revers" "kolonne" og "kolonne-revers"
